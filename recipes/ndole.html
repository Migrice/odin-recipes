Ndole

Description

Le ndolé est le nom camerounais de variétés alimentaires de Vernonia. Il s'agit d'un plat préparé à base d'une plante légumière dont les feuilles sont consommées vertes, mais aussi dans une moindre mesure séchées.

Ingredients

  • Les feuilles de ndole
  • Arachides
  • Oignons
  • Ail
  • Piment
  • Sel
  • Crevettes
  • Poivre
  • Huile Mayor

Recette

  1. Bien laver les feuilles de Ndolé après les avoir bouillies
  2. Tremper les arachides
  3. Enlever la peau et les écraser avec l'oignon, l’ail, le poivre et le piment.
  4. Une fois écrasés, il faut les verser dans la marmite pour la cuisson, laisser bien cuire les arachides.
  5. Mettre le ndole et bien tourner, ainsi que les crevettes écrasées
  6. Ajouter le sel
  7. Faire frire l'oignon dans l'huile et verser au dessus
